web : https://shayda.webcareproject.my.id/

Website e-commerce ini adalah platform belanja online untuk produk fashion muslimah yang dirancang dengan tampilan premium, elegan, dan mudah digunakan. Website ini mendukung proses belanja dari melihat katalog produk, memilih varian, checkout, pembayaran online, hingga pengiriman barang. Sistem juga dilengkapi dashboard admin untuk mengelola produk, pesanan, stok, pembayaran, dan pengiriman secara terpusat.

Fitur utama website ini meliputi:

Katalog Produk Fashion Muslimah
Menampilkan berbagai kategori produk seperti gamis, dress, tunik, blouse, outer, rok, celana, hijab, mukena, inner, one set, dan aksesoris.
Detail Produk Lengkap
Setiap produk memiliki foto, deskripsi, harga, stok, varian ukuran atau warna, berat, serta informasi dimensi untuk kebutuhan pengiriman.
Keranjang Belanja dan Wishlist
Customer dapat menyimpan produk favorit ke wishlist dan menambahkan produk ke keranjang sebelum melakukan checkout.
Checkout dan Manajemen Alamat
Customer dapat memilih alamat pengiriman, melihat ringkasan pesanan, ongkos kirim, total pembayaran, dan menyelesaikan transaksi dengan mudah.
Integrasi Payment Gateway Midtrans
Website mendukung pembayaran online menggunakan Midtrans, sehingga customer dapat membayar melalui berbagai metode pembayaran digital.
Integrasi Pengiriman Biteship
Sistem dapat menghitung ongkos kirim, memilih kurir, dan membantu proses pengiriman pesanan melalui integrasi Biteship.
Manajemen Pesanan Customer
Customer dapat melihat riwayat pesanan, detail pesanan, status pembayaran, status pengiriman, dan informasi tracking.
Dashboard Admin
Admin dapat mengelola produk, kategori, stok, pesanan, pembayaran, pengiriman, customer, dan informasi penting lainnya melalui halaman dashboard.
Notifikasi
Website menyediakan fitur notifikasi untuk memberikan informasi terkait status pesanan, pembayaran, pengiriman, atau aktivitas penting lainnya.
Autentikasi Customer
Website mendukung login dan register customer, termasuk opsi login menggunakan Google Auth.
